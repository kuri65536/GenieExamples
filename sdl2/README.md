# SDL2 with Genie

This example programs was created for Genie language
from the vala SDL example.

```
$ git clone this.repo
$ cd this.repo
$ make
```

## Prerequirement

- install valac
- install libsdl2-dev, libsdl2-gfx-dev, libsdl2-image-dev
